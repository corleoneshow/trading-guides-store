## Content Creation Workflow

### For Each Video:

1. **Pick topic from calendar**
2. **Generate script** (use the script generator)
3. **Record screen/charts** in TradingView
4. **Add AI voiceover** via ElevenLabs
5. **Edit in CapCut** - add captions, transitions
6. **Post at optimal time** (7-9am or 7-10pm)

### Hashtags to Use:
```
#optionstradingforbeginners #stockmarket #investing #tradingtips #options101 #financialeducation #learntoday #stockmarkettips #optionstrading #daytrader
```

### Optimal Posting Times:
- Morning: 7am - 9am
- Evening: 7pm - 10pm
- Best days: Tuesday, Thursday, Saturday
